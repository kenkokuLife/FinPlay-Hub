# Skill: Financial & M&A Logic

## 1. Professional Terminology
- Use **EBITDA**, **EBIT**, **CapEx**, **PP&E**, **Enterprise Value**.
- Avoid casual terms like "profit" or "spending".

## 2. P&L Structure
- **Comparison**: Always show "Yesterday Actual" vs "Today Projected".
- **Indicator Logic**: Green for positive delta, Red for negative, Yellow for significant CapEx.
