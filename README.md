# FinPlay-Hub

金融模拟实战教学平台。通过游戏化手段，帮助用户掌握 M&A、财报分析及企业估值核心技能。

## 🛠 技术栈
- **Framework**: Next.js (App Router)
- **Styling**: Tailwind CSS
- **Icons**: Lucide React
- **Logic**: Full-stack React with specialized UI/UX standards.

## 🤖 开发者助手使用指南 (AI Agent Usage)

本仓库包含 `agent-skills.md`，旨在为 Codex、Claude Code 或其他 AI 编程助手提供统一的 UI/UX 和系统设计准则。

### 如何让 AI 遵循此准则？

**场景 A：开始新页面/新功能开发**
在对话开始时，请发送以下指令：
> "请先读取项目根目录下的 `agent-skills.md`。接下来的开发必须严格遵守其中的深色模式规范、图标使用准则以及『灵魂链接』交互逻辑。"

**场景 B：代码审查 (Code Review)**
> "检查我当前的 UI 代码，是否符合 `agent-skills.md` 中的 Fintech 终端感要求？特别检查图标是否使用了 Lucide 以及间距是否符合网格规范。"
